Lugares
=======

* Inicialmente solo se puede ir a 4 lugares y según avanza el juego se van descubriendo nuevos lugares según recomendaciones en las citas

Javascript
----------

```javascript
[{
  key: string,      // Identificador del lugar
  value: string,    // Descripción
  hidden: boolean   // Oculto
}, {
  ...
}]
```

YAML
----

```yaml
Nombre: identificador
Nombre2: identificador?  # oculto con ? al final del identificador
```
